## January 12, 2026
*Creating my Own Project*

On this day, I started developing a small personal project to add to my portfolio. The project is called Timeshift, a watch sales website designed to showcase modern UI/UX concepts and front-end development skills.

The website was built using HTML, CSS, and JavaScript, focusing on clean layout structure, responsive design, and smooth user interactions. The goal of this project is to simulate an e-commerce landing page experience without using a backend or database, allowing me to focus purely on front-end logic and visual presentation.

During development, I worked on:

-Structuring semantic HTML for better accessibility
-Styling components with CSS for a modern and elegant look
-Adding interactivity and dynamic behavior using JavaScript
-Designing sections such as the hero area, featured watches, product highlights, and call-to-action components

This project helped me improve my understanding of front-end workflows, UI consistency, and creating visually appealing layouts suitable for real-world portfolio presentation.

## January 14, 2026
*Refining Design & Adding Product Interaction*

Today, I continued developing Timeshift, focusing on improving the user interface and adding interactive elements to make the website feel more complete and professional. My goal was to make the website not only look good but also feel smooth and engaging for users.

What I worked on today:

- Enhanced Product Cards
I improved the product card layout by adding clearer product details, price labels, and a more attractive presentation. I also added hover effects to make each card feel interactive and modern.

- Added Quick View Modal
To give users a better shopping experience, I added a quick view feature. When a user clicks on a product, a modal window appears showing more details, including product description, price, and options to add to cart.

- Improved Navigation & Header Design
I redesigned the navigation bar to be cleaner and easier to use. I also added a sticky header so users can easily access navigation even when scrolling.

- Responsive Enhancements
I tested the website on different screen sizes and made adjustments for mobile and tablet views. This included improving font sizes, spacing, and the layout of product grids.

- Added Smooth Animations
I added smooth animations using CSS and JavaScript to make transitions more appealing, such as fading in sections and sliding product cards.

Key Learning Today:

I learned how small details like hover effects, modals, and responsive spacing can significantly improve the overall user experience. I also gained better understanding of how to structure interactive UI components using JavaScript.
