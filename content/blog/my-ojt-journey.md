# April 6, 2026

Today, I started my task as a QA Tester for the final testing of the NCIP Online Job Application system. I carefully reviewed the system from the beginning, starting with the login and registration process. I tested if users could create accounts properly and if all input fields were working as expected. I also checked if error messages would appear when invalid data was entered. While testing, I noticed a few small issues in the validation, so I listed them down to report later. This day helped me understand how important it is to test even the smallest details in the system.

# April 7, 2026

I continued the final testing by focusing on the main features of the system, especially the job application process. I tested how users can browse job listings and submit their applications. I checked if all the buttons, forms, and file uploads were functioning correctly. I also made sure that the submitted applications were properly saved in the system. During the testing, I encountered minor bugs such as delays in loading and small UI issues, which I documented. This helped me improve my attention to detail as a QA Tester.

# April 8, 2026

Today, I tested the system more deeply by trying different user scenarios. I acted as different types of users to see how the system would respond in each situation. I checked if the system could handle incorrect inputs, incomplete forms, and repeated actions. I also tested the responsiveness of the system to make sure it works well on different screen sizes. I continued listing all the issues I found and organized them based on priority. This activity helped me understand how important it is to think like an actual user during testing.

# April 9, 2026

I focused on rechecking the issues that I previously reported to see if they were already fixed. I coordinated with the development team and tested the updated parts of the system. I made sure that the fixes did not affect other functions. I also reviewed the overall flow of the system again to ensure consistency and smooth navigation. This process helped me see how QA and developers work together to improve the system before release.

# April 10,2026 

On this day, I finalized the testing process of the NCIP Online Job Application system. I conducted a final run-through of all features to ensure everything was working properly. I double-checked all critical functions such as registration, job application, and data saving. I also made sure that the system was ready for deployment by confirming that all major issues were resolved. This experience as a QA Tester helped me gain confidence in testing systems and understanding the importance of quality assurance before releasing a project.![8fe08a6d-0392-45bb-a2b5-09a8aecb0af7.jpg](/blog-images/1777556987084-8fe08a6d-0392-45bb-a2b5-09a8aecb0af7.jpg)
